# Running

## Chuẩn bị

- <b>Cài đặt package </b>

 mở terminal và cd  tới backend và cd tới frontend sau đó gõ lệnh: 
  ``` 
  npm install
  ```

- Build source frontend: 

  ```
  npm run build
  ```

## Local (For DEV)

- Yêu cầu môi trường

  ```
  node v14+
  ```
